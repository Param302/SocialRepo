About

Social Repo is an Open Source browser extension which simplify your process of filling out forms and applications by storing your social media profile links.

Commands

Using commands, you can easily enter your social media profile link.
Typing a \<command> in input field will replace it with your relevant social media profile link.

Command Handle
\twitter Twitter
\github Github
\linkedin LinkedIn
